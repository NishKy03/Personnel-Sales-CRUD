# Personnel-Sales-CRUD
This project is a single-page web application to manage marketing personnel and their sales records.

## Tech Stack
- Backend: ASP.NET Core Web API (.NET 8)
- Frontend: HTML, Bootstrap, JavaScript
- Database: SQL Server

## Features
- Add, Edit, Delete Personnel
- View Sales by Personnel
- Pagination & Table Management
- RESTful API architecture

## Setup Instructions
Please refer to DeploymentGuide.pdf for full setup steps.

## Notes
- No authentication required
- Designed to run on IIS
